# Story Images

Letakkan 3 foto di folder ini dengan nama:

- `story-1.webp` — Chapter 01 (Awal Mula)
- `story-2.webp` — Chapter 02 (Resep Rahasia)
- `story-3.webp` — Chapter 03 (Untuk Semua)

**Format yang disarankan:**
- Format: `.webp` (lebih ringan & cepat)
- Resolusi: minimal 1280x800 (rasio 16:10) atau 1200x900 (rasio 4:3)
- Ukuran file: < 300KB per gambar (gunakan kompresi)

**Mau ganti nama/teks slide?**  
Edit data `STORY` di `lib/menu-data.ts`.
